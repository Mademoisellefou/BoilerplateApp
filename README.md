# Important

Create .env for example .env_EXAMPLE:

```
PORT=4000
REACT_APP_BACKEND_URL=http://localhost:4000
DOMAIN=melany.local
```

# Docs

Project for Melany

## Configuration
### Frontend
```
1.- cd frontend
2.- yarn install
```

### Backend
```
1.- cd server
2.- yarn install
```


### Hosts

nano etc/hosts
```
# Melany
127.0.1.1	melany.local
127.0.1.1	api.melany.local
```
