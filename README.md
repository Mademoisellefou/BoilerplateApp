# Docs

Project for Melany

## Configuration
### Frontend
```
1.- cd frontend
2.- yarn install
```

### Backend
```
1.- cd server
2.- yarn install
```


### Hosts

nano etc/hosts
```
# Melany
127.0.1.1	melany.local
127.0.1.1	api.melany.local
```
